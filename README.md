# 🧠 Memory Card Game

> *Challenge your memory. Match every pair. Have fun while improving your focus!*

A classic **Memory Card Game** built with **Python** where players flip hidden cards and match identical pairs. The game is designed to strengthen memory, concentration, and logical thinking while providing an enjoyable gaming experience.

---

## 🌟 Project Overview

The Memory Card Game is a simple yet engaging game that tests your ability to remember the position of hidden cards. Players reveal two cards at a time and try to find matching pairs. The challenge continues until every pair has been matched successfully.

This project was created to practice Python programming concepts by building a real-world interactive game.

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/vishalrana45/memory_card_game_py.git
```

2. Open the project folder

```bash
cd memory_card_game_py
```

3. Run the game

```bash
python memory_card_game.py
```
